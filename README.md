# LocalUp Homebrew Tap

Official Homebrew tap for localup-dev tools.

## Installation

First, tap the repository:

```bash
brew tap localup-dev/tap
```

Then install localup:

```bash
brew install localup
```

Or install directly:

```bash
brew install localup-dev/tap/localup
```

## Available Formulae

- **localup** - Local development environment setup and management

## Updating

To update to the latest version:

```bash
brew update
brew upgrade localup
```

## Uninstalling

```bash
brew uninstall localup
brew untap localup-dev/tap
```

## Development

To add or update formulae in this tap, please submit a pull request to this repository.
